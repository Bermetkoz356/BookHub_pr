### **Документ 1: Техническое задание (ТЗ) для книжной платформы**  
**Проект:** Goodreads-подобный веб-сайт для чтения книг  


---


#### **1. Цель проекта**  
Создать многофункциональную платформу для чтения, обсуждения и рекомендации книг с возможностью управления личной библиотекой.  


#### **2. Функциональные требования**  
**2.1. Пользовательские роли:**  
- **Гость:**  
  - Просмотр каталога книг  
  - Регистрация/авторизация (через email или OAuth)  
- **Пользователь:**  
  - Чтение книг онлайн (встроенный просмотрщик)  
  - Скачивание книг (PDF/EPUB)  
  - Управление списками: "Прочитано", "Читаю", "Буду читать"  
  - Комментирование, выставление рейтингов  
  - Подписки на других пользователей  
- **Администратор:**  
  - Добавление/удаление книг  
  - Модерация контента  
  - Доступ к аналитике  


**2.2. Основные страницы:**  
- **Главная:**  
  - Лента популярных книг  
  - Поиск по названию/автору/жанру  
- **Профиль пользователя:**  
  - Личная информация  
  - Вкладки с книгами, друзьями, настройками  
- **Страница книги:**  
  - Обложка, описание, отзывы  
  - Кнопки чтения/скачивания  
- **Раздел "Интересное":**  
  - Персонализированные рекомендации  


**2.3. Технические требования:**  
- **Frontend:** HTML/CSS (Bootstrap), JavaScript  
- **Backend:** Django (Python)  
- **База данных:** PostgreSQL  
- **Аутентификация:** Django Allauth  


---


#### **3. Нефункциональные требования**  
- Адаптивный дизайн (поддержка мобильных устройств)  
- Загрузка книг не более 2 секунд  
- Защита данных пользователей (GDPR)
